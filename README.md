# Arduino Smart Light System

## 專案介紹
本專案結合：
1.聲音感測器 (Microphone Sensor)
2.按鈕控制
3.光敏電阻 (LDR)
4.LED 燈光控制

主動觸發：使用者可以透過拍手或按鈕切換燈光開關。
被動觸發：光敏電阻（調節光線亮暗）

當系統開啟後：
光線較亮 → 亮 1 顆 LED
光線中等 → 亮 2 顆 LED
光線較暗 → 亮 3 顆 LED

## 使用元件
- Arduino UNO R3
- Microphone Sensor
- Push Button
- LDR 光敏電阻
- LED ×3

## 功能特色
1. 聲音觸發開關控制
2. 按鈕開關控制
3. 自動亮度調整
4. 即時序列埠監控
